# Principle

Tabula.md is a local-first Markdown workspace for files that people and coding
agents can share safely.

# Product Boundaries

- Keep Markdown files as the primary object.
- Do not turn the first screen into a dashboard, database, or marketing page.
- Treat the right panel as project context: Files, Outline, Comments.
- Treat comments as product comments, not separate notes.
- Use `Tabula.md` for product-facing naming and `Tabula` only as internal
  shorthand.
