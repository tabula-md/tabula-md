# Team Onboarding

Start here:

- Read [WORKFLOW.md](../WORKFLOW.md) for implementation, review, and merge.
- Read [AGENTS.md](../AGENTS.md) for project context and product constraints.
- Run `npm install`.
- Run `npm run workflow:status` before starting PR-bound work.

Do not maintain a separate workflow checklist in this file. `WORKFLOW.md` is the
standard.
