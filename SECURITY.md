# Security Policy

## Supported Versions

Tabula.md is in public preview. Security fixes target the latest code on
`main` until the project starts maintaining separate release lines.

## Reporting A Vulnerability

Do not open a public GitHub issue for security vulnerabilities.

Email security reports to:

```txt
security@tabula.md
```

Include:

- A clear description of the issue.
- Steps to reproduce.
- Affected URLs, versions, commits, or deployment context.
- Impact and any known exploitability.
- Whether the report can be publicly credited after a fix.

We aim to acknowledge reports within 72 hours.

## Disclosure

Please give maintainers a reasonable opportunity to investigate and release a
fix before public disclosure. We will coordinate credit and disclosure timing
with reporters whenever possible.
