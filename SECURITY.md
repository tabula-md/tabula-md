# Security

Please do not report security vulnerabilities in public issues.

Use GitHub's private vulnerability reporting for this repository:

https://github.com/tabula-md/tabula-md/security/advisories/new

Include the affected feature, reproduction steps, and potential impact. Do not
include active room or Export links unless they were created specifically for
the report and contain no private content.

Tabula.md is in public preview. Security fixes target the latest hosted version
and the current default branch.
