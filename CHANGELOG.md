# Changelog

All notable user-facing changes to Tabula.md are documented here.

This project follows semantic versioning before `1.0.0` with the usual
pre-1.0 caveat: minor versions may still include breaking product or storage
changes when the public preview needs them.

## v0.1.0 - 2026-06-18

Initial public preview.

### Added

- Local-first Markdown workspace with editable Markdown tabs.
- Markdown preview with GitHub Flavored Markdown support.
- File, outline, and comments project context panels.
- Experimental live Markdown sessions.
- Share flow for live room links.

### Notes

- Live collaboration is experimental in this preview.
- GitHub Issues are open for public bug reports and feature requests.
